watchFire
=========

NASA SpaceApps 2014 project

Modelo de datos
---------

*coordenadas: {type: "Point", coordinates: [longitude, latitude]},
*windSpeed: 5000,
*date: timestamp
*confidence: 0-100.0,
*temperature: 0.0-100.0,
*humidity: 0.0-100.0
*vegetation: -0.1-0.1
*frp: 0.0-300.0
*noise: 0.0-100.0
